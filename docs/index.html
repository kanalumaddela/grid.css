About

I only made this because a lot of my work involves sites related to garry's mod which uses Awesomium (Chrome 18). I made a minor specific fix, but then go to thinking if I made a framework, so it's easier. I wanted some serious browser compatibility along with the ability to opt for newer css features like flexbox, so I made this. I also wanted to allow different grids and added in a 10,12, and 16 column layout. This is a desktop first framework meaning that with something like 

<!-- full width on small, half width on medium, third width on large -->
<div class="col s12 m6 l4"></div>
the desktop width ( col l4 ) takes precedence on devices that don't support media queries.

Getting Started

  1. Download the latest version from the repo and place the files in your webspace
  2. Create your template page (best to remove comments) 
    <!DOCTYPE html>
<html lang="en">
<head>
    <title>My Page</title>
    <meta charset="utf-8">
    <!-- Mobile Support -->
    <meta name="viewport" content="width=device-width, initial-scale=1.0"/>
    <!-- Reduces DNS Lookups, optional, but helps if loading multiple assets from cdnjs -->
    <link rel="dns-prefetch" href="https://cdnjs.cloudflare.com/ajax/libs">
    <!-- normalize.css, required -->
    <link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/normalize/7.0.0/normalize.min.css" />
    <!-- grid.css, grid.slim preferred -->
    <link rel="stylesheet" href="path/to/css/grid.slim.min.css" />
</head>
<body>
    <div class="container">
        <!-- 12 grid system is preferred -->
        <div class="row grid-12">
            <!-- colums go here -->
        </div>
    </div>
</body>
</html>
  3. You'll notice there are 3 distinct css files to choose from (and their .min.css versions). Link the one that you feel would be best for your project.
    • grid - both grid.flex.css & grid.slim.css
    • grid.flex - uses flexbox for the grid layout, works on most modern browsers
    • grid.slim - uses display: inline-block for the grid layout, works on both old and modern browsers
